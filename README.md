# Rayfield Gen 2 [fanmade]

A fan made reimagining of the Rayfield UI library for Roblox Luau. Gen 2 keeps the
familiar clean look while rebuilding the internals for smoother performance and a
friendlier developer API.

> This is an unofficial community project. It is not affiliated with or endorsed by
> the original Rayfield authors.

## Status

Early work in progress. Expect the API to shift while the core comes together.

## Goals

* Faithful to the original Rayfield feel, rebuilt from scratch
* Lightweight and fast, no heavy asset hacks
* Simple, readable API for windows, tabs, and elements
* Clean theming so a UI can be restyled without touching layout code

## Planned elements

* Windows and tabs
* Buttons and toggles
* Sliders and inputs
* Dropdowns and keybinds
* Notifications
